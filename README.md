🚀 #Small Projects Collection

This repository is a collection of small frontend projects built during my learning journey in HTML, CSS, and JavaScript. Each project represents a step forward in understanding how these core technologies work together to create interactive and responsive web applications.

🎯 #Learning Objectives

Through these projects, I focused on:

Building a strong foundation in frontend development
Understanding how HTML structures, CSS styles, and JavaScript adds interactivity
Practicing real-world UI components and layouts
Improving problem-solving and debugging skills
Writing clean, readable, and maintainable code
Exploring DOM manipulation and event handling

💡 #Why This Repository?

Instead of just learning theory, I believe in learning by building. These projects helped me turn concepts into practical skills and gain confidence in creating real user interfaces from scratch.
